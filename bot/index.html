Nakano Nino

The best way to have fun and moderation on your server

Features

Economy

  • General You can recieve daily/weekly rewards, transfer money, work, commit crimes and steal from users!
  • Games There is Blackjack, Slot Machines, Dice Rolls, Connect 4 and Rock Paper Scissors to play!
  • Admin Admins can set, give and remove money from users!

Music Player

  • Music Channel Assign the Music Player Commands to only one channel
  • Search This will search YouTube for your request
  • YouTube You can use links from YouTube to play the videos or playlists you want
  • Spotify You can use links from Spotify to play the song or playlist you want
  • Apple Music You can use links from Apple Music to play the song or play like you want

GIFS

  • GIF This will get a random gif from your request and use it in the channel
  • Tag a user to make the bot send a gif to that user
  • Apply Multi Roles You can make the bot allow the user to select many roles
  • Hidden GIF There is a GIF that will send if you say a keyword (You will have to find that out)

Reaction Roles

  • Apply Reaction Roles to a Message ID Get a Message ID to set the Reaction Role to
  • Apply Single Role You can make the bot allow the user to only select one role at a time
  • Apply Multi Roles You can make the bot allow the user to select many roles

Verify Button

  • Verification By pressing the Verify Button, you will recieve a role
  • Title Change the title of the embed to be whatever you want
  • Description Change the description of the embed to be whatever you want
  • Role You can use any role on your server to give the user that role

Music Player

  • !play Shows name of the current track
  • !nowplaying Shows name of the current track
  • !skip Skips the current track
  • !pause Pauses the current track
  • !resume Resumes the current track
  • !loop Toggles the loops of the current track
  • !queue Shows the queue
  • !clearqueue Clears the queue
  • !remove < queue id > Removes a track from the queue
  • !stop Stops the queue
  • !setvolume Sets bot volume
  • !getvolume Shows bot volume
  • !join Joins voice channel
  • !leave Leaves voice channel

Economy

  • General
  • !bal Shows your balance
  • !daily Redeem daily £100
  • !weekly Redeem weekly £1000
  • !work Get a random job to work for and get money (Available every 4 hours)
  • !crime A 20% chance of commiting a crime to get money (Available every 6 hours)
  • !steal < user > A 5% chance of stealing half of the users money, or losing it all (Available every 6 hours)
  • !transfer < user > < ammount > - Transfers to the user the ammount you said

  • Games
  • !bj < bet ammount > Play blackjack with a bet you want to get a chance to double your bet
  • !roll < bet ammount > Roll a dice with a bet you want to get a chance to double your bet
  • !slots < bet ammount > Play at the slot machine with a bet you want to get a chance to double your bet
  • !connect4 < user > < bet ammount > Play Connect 4 against another user to get a chance to get the bet ammount from the other user
  • !rps < user > < bet ammount > Play Rock Paper Scissors against another user to get a chance to get the bet ammount from the other user
  • !roulette - Comming Soon!

  • Admin
  • !bal < user > Shows balance of the user you said
  • !setmoney < user > < ammount > Sets the ammount to the user you said
  • !givemoney < user > < ammount > Gives the ammount to the user you said
  • !removemoney < user > < ammount > Removes the ammount to the user you said

Reaction Roles

  • !rrcreate < msg id > < single/multi > < emoji > < role > Creates a Reaction Role

  • < msg id > The id of the message you want to add reaction roles to
  • < single/multi > Sets the reaction role to allow singular/multiple reactions
  • < emoji > The emoji/custom emoji you want to add
  • < role > Assigns the role with the emoji, tag the role or role id
    To add more reaction roles, repeat < emoji > < role >

  • !rredit < msg id > < single/multi > < emoji > < role > Edits a Reaction Role (removes all reactions and reapplies)

  • < msg id > The id of the message you want to edit reaction roles of
  • < single/multi > Change the reaction role to allow singular/multiple reactions
  • < emoji > The emoji/custom emoji you want to add
  • < role > Assigns the role with the emoji, tag the role or role id
    To add more reaction roles, repeat < emoji > < role >

  • !rrdelete < msg id > Deletes a Reaction Role

  • < msg id > The id of the message you want to delete reaction roles from
